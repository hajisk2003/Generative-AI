# LangChain Output Parsers

This folder contains my hands-on practice with different output parsers in LangChain.  
The main goal was to understand how LLM responses can be converted into structured and usable formats instead of plain text.

---

## 📌 What are Output Parsers?

Output parsers are used to format and structure the response generated by LLMs.  
Instead of getting raw text, we can convert responses into:

- Plain strings
- JSON format
- Structured schema-based output
- Pydantic objects

This makes LLM responses easier to use in real-world applications.

---

# 📂 Files Overview

## 🔹 stringOutputParser.py

### Definition

Converts LLM output into a clean plain string.

### Implementation

- Used `StrOutputParser`
- Removed unnecessary metadata
- Returned only text response

### Use Case

- Simple chatbots
- Basic Q&A systems
- Text generation tasks
