# LangChain Practice – GenAI Learning

This repository contains my hands-on practice while learning Generative AI using LangChain.  
I’m following a structured approach where I explore concepts and implement small working examples.

---

## 📌 What’s Covered

### 🔹 Prompt Handling

- Basic prompt templates
- Structured input using `PromptTemplate`
- Dynamic prompt generation

### 🔹 Chat Models

- Difference between LLM and Chat Models
- Using `ChatPromptTemplate`
- Message roles (System, Human, AI)

### 🔹 Stateful Chatbot

- Built a simple chatbot with memory
- Stored chat history in a file (`chat_history.txt`)
- Maintained conversation context across interactions

### 🔹 Message Handling

- Used message placeholders
- Managed conversation flow using LangChain message objects

### 🔹 Temperature Control

- Tested how temperature affects LLM responses
- Compared deterministic vs creative outputs
